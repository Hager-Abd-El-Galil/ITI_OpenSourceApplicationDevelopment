Bloggerwp theme is lightweight Responsive theme.

== Description ==

Bloggerwp theme is lightweight Responsive theme. it supports Gutenberg Editor, custom logo, menu, page, post, custom background color , widgets , custom copyright text

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use bloggerwp theme.
